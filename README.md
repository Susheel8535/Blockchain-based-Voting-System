# 🗳️ Blockchain Voting System

A decentralized voting application built using **Ethereum, Hardhat, and Next.js** that ensures secure, transparent, and tamper-proof elections.

---

## 🚀 Features

* 🔐 **Secure Voting** using smart contracts
* 🧾 **Transparent Results** stored on blockchain
* 👤 **Voter Registration System**
* 🧑‍💼 **Candidate Management**
* 🗳️ **One Person, One Vote** mechanism
* 🌐 **Web3 Integration** with MetaMask

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React.js
* **Blockchain:** Ethereum, Hardhat
* **Smart Contracts:** Solidity
* **Web3 Integration:** Ethers.js, Web3Modal
* **Storage:** IPFS (Pinata)

---

## 📂 Project Structure

```
├── components/       # UI Components (Card, NavBar, VoterCard)
├── context/          # Voting Context (Blockchain logic)
├── pages/            # Next.js Pages
├── scripts/          # Smart contract deployment scripts
├── styles/           # CSS Modules
├── contracts/        # Solidity smart contracts
```

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/blockchain-voting-system.git
cd blockchain-voting-system
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Start Hardhat local blockchain

```bash
npx hardhat node
```

---

### 4️⃣ Deploy smart contract

```bash
npx hardhat run scripts/deploy.js --network localhost
```

---

### 5️⃣ Update contract address

Update the deployed contract address in:

```
context/constants.js
```

---

### 6️⃣ Run frontend

```bash
npm run dev
```

---

## 🦊 MetaMask Setup

* Network Name: Localhost
* RPC URL: http://127.0.0.1:8545
* Chain ID: 31337
* Import Hardhat account using private key
